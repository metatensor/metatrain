Symplectic FlashMD (Experimental)
=================================

.. image:: https://codecov.io/gh/metatensor/metatrain/branch/main/graph/badge.svg?component=flashmd_symplectic
   :target: https://app.codecov.io/gh/metatensor/metatrain/tree/main/src/metatrain/experimental/flashmd_symplectic?components%5B0%5D=flashmd_symplectic

*Maintained by* `@frostedoyster <https://github.com/frostedoyster>`_, `@johannes-spies <https://github.com/johannes-spies>`_.

The symplectic variant of :ref:`FlashMD <arch-flashmd>`.

.. _arch-flashmd_symplectic_installation:

Installation
------------

To install this architecture along with the ``metatrain`` package, run:

.. code-block:: bash

    pip install metatrain[flashmd_symplectic]

where the square brackets indicate that you want to install the optional
dependencies required for ``flashmd_symplectic``.
